# weliton

## Entre Cicatrizes e Versos

Uma biblioteca digital criada para reunir todos os poemas de Weliton.

Este projeto nasceu com um propósito simples:

Dar um lar para cada poema.

Ao invés de redes sociais repletas de distrações, aqui existe apenas uma coisa:

Poesia.

## Estrutura

- Página Inicial
- Biblioteca organizada por sentimentos
- Página Sobre
- Poemas carregados através de JSON

## Tecnologias

- HTML5
- CSS3
- JavaScript
